Plugin para mostrar el ultimo sismo reportado por SNET en El Salvador.

== Description ==

Este plugin mostrara una imagen estatica de Google Maps con un marcador donde ocurrio el ultimo sismo reportado por el por SNET en El Salvador.

**Features**

* Ridiculamente sencillo de usar.

== Frequently Asked Questions ==
* ¿Es la información mostrada en tiempo real?
    NO. Se muestra hasta que SismoSV la ha obtenido de SNET, y SNET tiene una "ventana" de tiempo de publicación de información de 20 minutos aproximadamente. Así que esta información en NINGUN momento se debe pensar como una alerta.
    
* ¿Cada cuanto cambia la información mostrada?
    Cada vez que hay un sismo y se publica en la página oficial de SNET.
